# Extract_package
팀 5Zigo-Gri-jo(5지고 지리조)의 Extract 모듈 레포지토리입니다.


## 설치방법
- repository 설치 방법
```
git clone https://github.com/5Zigo-Gri-jo/Extract.git
```

- 설치 이후 환경설정
```
pdm init
pdm install
source .venv/bin/activate
```

## 모듈 기능
### extract_package.py
* 영화진흥위원회 오픈 API 기능을 통해 지정한 년도의 박스오피스 데이터를 ``` request ```받아 저장하는 모듈입니다.
